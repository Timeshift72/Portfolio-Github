Gestion de projet web

Tout commence par la définition des objectifs et s'y engager en prenant appui sur les "users stories".
Méthode Agile et SCRUM avec l'équipe.

Editeur de texte

Utilisation de Visual Studio Code.
Sublime Text 3.
Bien respecter la syntaxe et l'indentation.
Bien commenter les différentes sections.

Options affinées

Bien choisir ses options d'habillage avec Bootstrap par exemple et un peu de code javascript pour le rendu dynamique.
Où bien du PHP avec MySQL par exemple pour une base de données.

Outil de gestion en ligne

Planifiez les rendez-vous, réunions et évaluations périodiques à l'aide du logiciel de gestion Trello, Asana.

Navigation

Lecture facile sur les différents navigateurs Chrome, Edge, Firefox, Opéra, Safari.

Bonne fondation

Obtenez une base solide pour les créations suivant les bonnes pratiques, en adéquation avec le besoin et en ayant une structure pérenne et organisée.

Suivi des résultats en fonction de votre stratégie personnel ou d'équipe.

Une fois la visualisation globale du site avec ses "users stories" déterminée, on s'interessera aux détails intéressants. Attention le projet n'est jamais statique, vous pouvez le changer.

PROTOTYPAGE

Un bon croquis de base est parfois suffisant pour un projet, mais pour des projets plus conséquant il faut bien diviser les fonctions, l'affichage et prévoir chaque liaison en utilisant un "wireframe".
